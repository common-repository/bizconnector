A BizConnector Product

== Description ==

The bizconnector plugin is intended for use by BizConnector customers. BizConnector / Lead Follow-Up is
a marketing automation app on Salesforce.com (https://appexchange.salesforce.com/listingDetail?listingId=a0N300000016Y29EAE)

This plugin enables blog post and page content to be displayed in email messages. It also enables rules
that can fire when new blog posts are created. See http://www.bizconnector.com for more information.

== Frequently Asked Questions ==

= Can I test this plugin during my free trial? =

Yes.

= Why is this plugin useful? =

Because it automates the sending of emails or texts when you create new blog posts. You can set up rules that send only to people who match certain conditions. eg. they have signed up to receive notifications when you write a new blog post.

== Installation ==

1. Upload the `bizconnector` folder to the `/wp-content/plugins/` directory or install directly through the plugin installer.
2. Activate the plugin through the 'Plugins' menu in WordPress or by using the link provided by the plugin installer.

== Documentation ==

BizConnector is a marketing and workflow automation app on Salesforce.com, aimed at small businesses.

Automation is enabled through business rules that you create. 

For example:
- "Start a drip sequence of relevant emails to a new lead"
- "Send SMS (text) messages to interested contacts when you write a new blog post"

See http://www.bizconnector.com for more information. 

You can sign up for a free trial at https://appexchange.salesforce.com/listingDetail?listingId=a0N300000016Y29EAE - click the "Get It Now" button.

The plugin is available at http://www.bizconnector.com/wordpress/plugin/bizconnector.zip

This plugin uses code from JSON API - a Wordpress plugin written by Dan Phiffer. Permission obtained (October 2013) from Dan Phiffer to do this.
